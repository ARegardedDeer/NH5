## Folder Structure

- `src/app`: Entry screens, navigation, and bootstrap for the React Native app shell.
- `src/config`: Runtime configuration helpers (environment flags, constants, feature toggles).
- `src/db`: Database access utilities and Supabase client setup.
- `src/state`: Shared state management (stores, reducers, atoms).
- `src/features/anime`: Anime-related feature modules, UIs, and logic.
- `src/features/profile`: Profile management flows and components.
- `src/features/quests`: Quest-related features, data flows, and screens.
- `src/ui/components`: Reusable UI building blocks shared across features.
- `docs`: Project documentation and reference materials.
